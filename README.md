# 💰 Expense Tracker

**Expense Tracker** is a simple Python-based command-line application that helps users manage their daily expenses efficiently. It allows users to add, view, edit, delete, and search expenses while storing all records permanently in a CSV file. The project also provides category-wise expense reports, making it easier to analyze spending habits.

## ✨ Features

* ➕ Add new expenses
* 📋 View all expenses
* 💰 Calculate total expenses
* ✏️ Edit existing expenses
* 🗑️ Delete expenses
* 🔍 Search expenses by category
* 📊 Category-wise expense report
* 💾 Automatic data storage using CSV files

## 🛠️ Technologies Used

* Python
* CSV Module
* File Handling
* Lists & Dictionaries
* Exception Handling

## 🎯 Purpose

This project was developed as part of my **Python Summer Internship** to strengthen my understanding of Python fundamentals, file handling, and CRUD (Create, Read, Update, Delete) operations through a real-world application.
